The DegradationPreferences enum.

Based on the spec in https://w3c.github.io/webrtc-pc/#idl-def-rtcdegradationpreference.

< Don't take any actions based on over-utilization signals.

< On over-use, request lower frame rate, possibly causing frame drops.

< On over-use, request lower resolution, possibly causing down-scaling.

< Try to strike a "pleasing" balance between frame rate or resolution.

< No degradation preference, let the SDK decide how to handle over-utilization.
