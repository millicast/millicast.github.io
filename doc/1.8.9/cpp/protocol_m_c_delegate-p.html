Detailed Description

The MCDelegate protocol contains common methods that will be called on specific events from a Client object. It is mainly subclassed by MCSubscriberDelegate and MCPublisherDelegate, which are the main ones to implement.

◆ onStatsReport:

- (void) onStatsReport: (nonnull MCStatsReport *)  report

Called when a new RTC statistics report has been collected. You must enable statistics via MCClient/enableStats:completionHandler: to start receiving this handler.
