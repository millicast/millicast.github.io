This playback controller allows you to modify the state of the player node that is provided through the delegate.

Do not attempt to schedule/start/stop playback of the AVAudioPlayerNode directly. Use this controller instead. Request the AVAudioPlayerNode to start playback and receive audio.

Delegate that allows clients to tap into the AVAudioPlayerNode responsible for playback the SDK audio.

Notifies when the SDK creates an AVAudioPlayerNode that will be used for receiving audio -Parameters:
Global Audio Configuration object. Delegate for receiving AVAudioPlayerNode updates.

The AVAudioPlayerNode receives the SDK audio and can be routed to your own audio pipeline. Once the delegate is registered, the SDK will not handle audio playback internally.
Set the stereo pan if applicable. This property will apply panning if the played audio is in stereo. Valid values are -1 to 1
